命令行修改IP地址

    这篇文章讲解了怎样在命令行下，修改IP地址。

    netsh interface ip set address “以太网” static 192.168.0.100 255.255.255.0 192.168.0.1

    注：”以太网”是你网络的名称，引号为中文双引号，192.168.0.100为你想要修改的IP，255.255.255.0为子网掩码，192.168.0.1为默认网关
