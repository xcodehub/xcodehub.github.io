Kali Linux下安装fcitx中文输入法

    Kali Linux下安装fcitx中文输入法

    安装linux操作系统选择的是英文版本，但毕竟还是要输入中文，于是开始安装中文输入法，选择的是fcitx。网上很多说可以安装搜狗和谷歌输入法的但是尝试多次都是失败的。

    ###安装fcitx输入法

    1、在终端输入以下命令

    apt-get clean && apt-get update -y #首先更新一下源
sudo apt-get install "fcitx"
sudo apt-get install fcitx-pinyin

    这只是安装最基本的中文输入法，还可以安装谷歌或搜狗的输入法，但是我现在用的kali linux 2019.4安装谷歌和搜狗时会出错，我也不知道怎样解决。所以我只是安装了最基本的输入法。

    ###安装搜狗输入法 1、安装fcitx

    apt-get install fcitx

    顺利的话会自动安装成功，如果遇到依赖错误，按提示修复或者输入apt-get -f install 修复，但是如果提示“依赖xxx，但是xxx将不会被安装”，这一般是源太旧，建议换个其他的源。

    2、安装fcitx-libs-qt也许不需要

    apt-get install fcitx-libs-qt

    3、下载安装搜狗输入法 去搜狗输入法官网http://pinyin.sogou.com/linux/ 下载自己系统对应版本的输入法安装包。到下载的搜狗输入法安装包目录。然后运行

    dpkg -i 下载的sogo安装包文件名

    这一步比较慢，如果提示有依赖没有安装，可以用apt-get –fix-broken install （ 或apt –fix-broken insatll 或apt-get -f install）安装，如果提示“sogoupinyin依赖xxx(>=1.2.3…)，但系统中版本为1.1.1”之类的，说明源太旧了，需要换个其他源。可以用apt-get remove [package-name] 卸载旧版软件。要查看制定软件可用版本（本机的源上）可以用apt-cache madison [package-name] 查看，要安装制定版本的软件，可以用apt-get install fcitx=1.2.3 这样的语法安装。

    4、启用搜狗输入法 终端中输入：

    im-config

    进入输入法配置，选择确定，然后点击是，在弹出来的输入法选择框中选择fcitx，然后确定 。 注销账号，重新登录，然后右上角有个键盘图标，点击选择【设置】，在左下角点击【+】新增输入法 在弹出窗口中取消“仅显示当前预览”复选框，然后搜索Sogou,找到搜狗输入法，点击OK。打开编辑器，“Ctrl+，”即可切换到搜狗输入法

    ###安装谷歌输入 首先也要安装fcitx。

    apt-get install fcitx-googlepinyin
