解决kali linux中文乱码

    安装KALI2019.4版本后会出现乱码问题，即使选择引文版时浏览中文网页同样会出现乱码。

    1、安装KALI2019.4版本后会出现乱码问题

    2、更改安装源

    在解决之前可以先更改成中国安装源，这样是为了从安装源下载软件包会快一些，下面时吧kali在解决之前可以先更改成中国安装源，这样是为了从安装源下载软件包会快一些，下面时吧kali linux的安装源改为清华大学的

    #清华大学 [更新源]
deb https://mirrors.tuna.tsinghua.edu.cn/kali kali-rolling main contrib non-free
 
deb-src https://mirrors.tuna.tsinghua.edu.cn/kali kali-rolling main contrib non-free

    3、上面编辑完成然后使用执行下面命令

    apt-get clean && apt-get update -y

    4、在执行下面安装字体

    sudo apt-get install ttf-wqy-microhei ttf-wqy-zenhei xfonts-intl-chinese

    5、更改默认的编码，

    命令窗口输入dpkg-reconfigure locales进入图形界面，选中en_US.UTF-8 UTF-8和zh_CN.UTF-8 UTF-8并将zh_CN.UTF-8 UTF-8选为默认

    dpkg-reconfigure locales

    6、reboot重启过后，恢复正常
